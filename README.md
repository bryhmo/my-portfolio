## 🚀 Getting Started

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/bryhmo/my-portfolio

# Open in browser
cd my-portfolio
open index.html
```

Or simply open `index.html` in any browser — no build tools or installation needed. All three files (`index.html`, `style.css`, `script.js`) must be in the same folder.

## 💳 Payment Integration

The portfolio includes a payment section powered by **Flutterwave** and **Paystack** for clients to pay directly in Nigerian Naira (₦).

To activate payments, replace the placeholder keys in `script.js`:
```javascript
// Flutterwave
public_key: "YOUR_FLUTTERWAVE_PUBLIC_KEY"

// Paystack
key: "YOUR_PAYSTACK_PUBLIC_KEY"
```

Get your keys at:
- [Flutterwave Dashboard](https://dashboard.flutterwave.com)
- [Paystack Dashboard](https://dashboard.paystack.com)
